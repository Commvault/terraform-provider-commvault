# commvault_awsprotectiongroup (Resource)

Use the commvault_awsprotectiongroup resource type to create, update or delete an AWS Protection Group in the CommCell environment. 

## Example Usage

**Create Protection Group for RDS with all accounts and tag based content**

```hcl
resource "commvault_awsprotectiongroup" "example_pg_1" {
  name    = “PG-RDS”
  allcloudaccounts = "true"        

  # If the protection group is intended to be tied to the cloud connection created above:
 
  cloudconnection {
    name = “AWSOrgConnection”
  }

  # Plan block 
  plan {
    
    name = “RDS-Plan”
    # name can be added if needed, e.g. name = "Default Plan"
  }


  # content -> one content block with matchRule ALL and one TAG rule (value "true")
  content {
    matchrule   = "ALL"
    description = "Auto-generated from user input"

    rules {
      type      = "TAG"
      condition = "EQUALS"
      name      = "backup"
      value	= "true"
    
    }
  }

  # workloads -> single workload with name "Aurora and RDS Snapshot"
  workloads {
    workload {
      name = "Aurora and RDS Snapshot"
      id = 8004
     #####################
     # ForDocumentDB	
     # id = 8027
     # name = DocumentDB
     # ForEC2
     # id = 10301
     # name = EC2
     # ForEFS	
     # id = 6023
     # name = EFS
     #####################
    }
  }

}
```
